# Blackbox-testing
 
# Black Box Testing - Bài Thực Hành 03

## 📋 Giới thiệu
Repository này chứa **8 chương trình Python** được viết để giải quyết các bài toán cơ bản theo đề bài "Bài thực hành 03 - Kiểm thử hộp đen".

Mục tiêu chính là áp dụng kỹ thuật **kiểm thử hộp đen (Black-box Testing)**, bao gồm:
- Phân lớp tương đương (Equivalence Partitioning)
- Phân tích giá trị biên (Boundary Value Analysis)
- Kiểm tra dữ liệu hợp lệ và không hợp lệ
